# Changelog

## [0.1.0] - Initial Release

### Added
- Todo management with CRUD operations
- Priority levels (1-10) with customizable names and colors
- Categories for organizing todos
- Tags for flexible labeling
- Subtasks support (full todos as children)
- File attachments
- Due dates with time
- Web UI with dark theme
- REST API for Home Assistant integration
- Ingress support for seamless HA integration
